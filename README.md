# Shanti Recovery — Mobile App UI

A complete, interactive UI prototype for the **Shanti Recovery Wellness** mobile app, built as self-contained HTML.

**▶ Live preview:** https://ravi26design.github.io/shanti-recovery-app/

## What's inside
- **`index.html`** — the complete app flow (splash → auth → assessment → home → all 13 feature flows), 39 linked screens, on the Shanti brand palette (rose `#C8A297` / charcoal `#333132`) with the real logo. Includes a screen-jump menu for review.
- `shanti-app.html` — same as index.
- `shanti-prototype.html`, `shanti-login.html`, `shanti-ui.html` — earlier design iterations.

Faithful to the Figma wireframes, recolored to the brand and elevated to production-grade mobile UI (iOS status bar, dynamic-island frame, depth, motion).
